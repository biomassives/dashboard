About Studio Dashboard

The goal of this project is to create an admin dashboard for managing nodes in a cluster. It's currently focused on the specific use case of managing the SUSE Studio cluster, but we hope that it will be generic enough to be useful for others as well. The dashboard will pull key information/metrics from other systems and also allow us to store/track metadata, such as comments, warranty information, etc.

Studio Dashboard was born from a project in SUSE's 9th Hackweek, similar to how Team Dashboard started too (from Xing sponsored innovation week).
